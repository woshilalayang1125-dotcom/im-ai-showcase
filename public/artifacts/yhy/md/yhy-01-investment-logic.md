# 扬贺扬微电子投资逻辑框架_自然高密度优化v5

适用位置：扬贺扬微电子立项报告 / 后续 CATL 风格图片生成与可编辑PPT重建。  
核心要求：报告基本框架保持为 **投资逻辑 -> 行业分析 -> 公司分析 -> 财务、盈利预测与风险提示**；投资逻辑必须严格沿用《扬贺扬投资逻辑及要点阐述_优化版_20260527》的四条主逻辑，行业和公司章节分别展开四条主逻辑下的小支撑逻辑。

---

## 0. v5修订原则

1. **投资逻辑页只讲一件事**：用一页讲清四大逻辑及每条逻辑下的小支撑点，不再把投资逻辑拆散到多页。
2. **四大逻辑严格来自优化版PDF**：EOL供给真空、SLC窗口型标的、MLC中期alpha、跨区域团队与供应链协同，不新增“宁德协同”等无材料支撑逻辑。
3. **行业与公司双章节分别承接四大逻辑**：行业讲“窗口为什么存在、窗口需要什么样的企业”；公司讲“扬贺扬是否具备对应产品、客户、技术、团队和供应链能力”。
4. **标题必须是观点句**：每页标题表达因果、判断或投资结论，不使用“市场规模分析”“分场景看”“公司基本情况”等事实标签。
5. **证据分层使用**：客户访谈、财务表、客户穿透、公司答复和尽调报告作为核心证据；公司PPT与管理层预测需标注口径；MLC相关内容不得按已兑现处理。

## 1. 一句话投资判断

扬贺扬不是“AI直接拉动2D NAND需求”的标的，而是 **AI/HBM/企业级SSD周期抬高海外大厂成熟2D NAND机会成本后，SLC/MLC成熟平台被EOL形成的国产替代窗口型标的**：短期看16/19nm SLC在中兴/网通链条的放量与涨价兑现，中期看MLC/eMMC若完成流片、良率、可靠性、客户认证和订单兑现后，估值逻辑可从缺货周期受益上移到嵌入式存储方案平台。

## 3. 行业章节主线

行业章节不是泛存储介绍，而是按四大逻辑回答四个问题：

| 行业问题 | 对应四大逻辑 | 需要证明的结论 |
|---|---|---|
| 为什么现在有窗口 | 逻辑一 | AI存储高回报把成熟2D NAND挤出海外大厂资源表，EOL进入可排期窗口，客户不能等到断供后再替换 |
| 窗口是否足够大且可持续 | 逻辑一、二 | 2D NAND需求收缩到小容量高可靠长生命周期场景；SLC/MLC合计仍是百亿美元级利基市场，中国市场足以支撑少数细分国产厂商 |
| 窗口的价值结构在哪里 | 逻辑二、三 | 网通贡献数量和率先放量，工控/车载贡献粘性和毛利质量，MLC/eMMC贡献容量升级和中期alpha |
| 什么企业具备抓窗口画像 | 逻辑三、四 | 只有具备量产产品、客户认证、稳定供货、主控固件、CP测试、可靠性和跨区域工程团队的公司，才能把价格窗口变成长期份额 |

| 公司验证问题          | 对应行业结论             | 可用证据                                                       | 证据边界                           |
| --------------- | ------------------ | ---------------------------------------------------------- | ------------------------------ |
| 公司是否已经有可交付SLC产品 | 客户二供认证需要窗口前已有产品    | 16/19nm SLC量产；1Gb-8Gb；3.3V/1.8V；SPI NAND                   | 需补良率、坏块率、P/E、数据保持、PPM、退货率      |
| 客户是否真实导入且愿意承受涨价 | 2D NAND小容量高可靠需求仍刚性 | 中兴访谈：光猫/路由器/摄像头，1Gb/2Gb/4Gb，0.8-1.0亿颗/年，导入半年到一年，价格数倍上涨     | 需区分终端design-in、分销出货和渠道库存       |

## 5. 财务、估值与风险收束

| 模块 | v5口径 |
|---|---|
| 财务预测 | 不线性外推2026Q1高景气，把增长拆成SLC ASP、销量/产能、MLC/eMMC兑现三变量 |
| 估值 | 投前50亿元拆成SLC周期利润、MLC期权、平台化溢价三层；MLC和平台化价值必须按里程碑释放 |
| 核心风险 | 最大风险不是窗口不存在，而是估值提前计满周期价格、客户粘性和MLC期权 |
| 投资建议 | 建议有条件推进；交割前必须锁定SLC质量/客户、MLC、供应链、财务质量四类硬证据 |

## 6. 后续生成PPT的标题规则

1. 标题必须能回答“所以呢”，避免事实标签。
2. 标题优先使用“因果/转折/判断/估值含义”结构，例如“AI存储高回报正在把成熟2D NAND挤出大厂资源表”。
3. 禁用标题类型：“XX分析”“XX情况”“XX拆解”“XX基本情况”“SLC窗口的价值要按下游拆开看”。
4. 每页只放一个完整观点；相同逻辑尽量一页讲完，避免把一条证明链切碎。
